Retweeters Graph

About this

Twitter's REST API doesn't provides the data of the path a retweet followed to reach each of its retweeters. So that we need to do a deep search in all the followers of the User that created the original Tweet to build this graph.

An example Tweet URL: https://twitter.com/Le_Ruvb/status/638394608746135553

This can take SEVERAL minutes depending on the number of followers of each Retweeter and how deep the following chain is. This is because of the Twitter's API rate limits.
